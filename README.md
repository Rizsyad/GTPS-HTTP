# GTPS HTTP

**simple http, feel free for use it**

## Features

- Anti DDos
- Custome Ip and Port udp

## Required

- [Node.js](https://nodejs.org/en/)
- [pkg](https://www.npmjs.com/package/pkg) (npm install -g pkg)

## How To Use

- run

```
$ yarn install
$ yarn run start
```

or

```
npm install
npm run start
```
