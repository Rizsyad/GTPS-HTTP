# GTPS HTTP

**simple http, feel free for use it**

## Features

- Anti DDos
- Custome Ip and Port udp

## Required

- [Node.js](https://nodejs.org/en/)
- [pkg](https://www.npmjs.com/package/pkg) (npm install -g pkg)

## How To Use

```
$ yarn install
$ yarn run start
```

or

```
$ npm install
$ npm run start
```

## Testing DDOS

```
$ yarn run ddos
```

or

```
$ npm run ddos
```

## Convert to exe

```
$ yarn run buildy
```

or

```
$ npm run buildn
```
